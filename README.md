## Quick Overview

- This is a server of a tools manufacturer company who makes tools such as power saw, drill machine, electric saw, stapler machine

# this server contains code for

- get all products at once
- get single product at once
- get orders details of a user
- get all orders details for admin
- get stripe client secret key
- get reviews for home page
- storing reviews, products, user information
- updating user information, order information

## Technology Used in this site

- Node JS
- Express JS
- MongoDB database
- Stripe Payment Service
- Heroku
- dotenv
- cors
